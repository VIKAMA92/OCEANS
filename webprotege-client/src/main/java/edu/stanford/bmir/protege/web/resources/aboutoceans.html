About OntoCommons and OCEANS

What is OntoCommons?

OntoCommons is an H2020 CSA project dedicated to the standardisation of data documentation across all domains related to materials and manufacturing.

What is OCEANS?

OCEANS is OntoCommons Collaborative Editing and Authoring kNowledge base/graphS. The underlying formalisation of OCEANS is a lightweight ontology in OWL, and the platform used for the collaborative authoring is a customisation of WebProtégé. This platform supports OntoCommons consortium and community based authoring of OCEANS on the Web and provides features such as discussion threads integrated in the authoring process, change tracking, content reviewing, and so on.
